> Day.dart is inspired by Day.js. Write with nearly the same API. Build on the top of the powerful **DateTime** class.

* 🕒 Nearly the same API as Day.js
* 💪 Contains Immutable Methods
* 🔗 Chainable
* 👀 Useful formatting method
